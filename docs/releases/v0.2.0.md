# TreeRAG v0.2.0

TreeRAG v0.2.0 sharpens the retrieval story around two themes: answers are easier to trace back to the source, and the repo now includes a more realistic evaluation pack for appendix-heavy professional documents.

## Highlights

- Added source-traceable retrieval metadata for indexed documents.
- Exposed `source_path`, `selected_source_span`, and `source_references` in query outputs.
- Preserved source spans through index serialization and loading.
- Added a finance-style appendix benchmark fixture for low-overlap retrieval questions.
- Documented the appendix evaluation flow in the README and manual QA checklist.
